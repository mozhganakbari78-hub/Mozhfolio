Hero section images go here.

Upload the character image into this folder (public/hero/) and name it:
  hero-character.png

Anything inside /public is served from the site root, so this file becomes
available at:  /hero/hero-character.png

Steps to upload on GitHub:
1. Open this folder (public/hero) on github.com
2. Click "Add file" -> "Upload files"
3. Drag the PNG in, rename it to hero-character.png if needed
4. Commit to the branch claude/compassionate-albattani-2xv7ko
